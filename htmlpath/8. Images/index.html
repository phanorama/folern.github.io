Selamat datang di HTML
Pendahuluan
HTML adalah singkatan dari Hypertext Markup Language, yaitu bahasa markup standar untuk membuat dan menyusun halaman dan aplikasi web.

Kamu bisa menggunakan bahasa markup ini untuk bagian paragraf, heading, maupun link pada suatu web page.

Nah, dalam Web ini, kita akan membahas semua hal tentang HTML, termasuk apa itu HTML, pengertian HTML, dan lain sebagainya.

Selamat belajar!
